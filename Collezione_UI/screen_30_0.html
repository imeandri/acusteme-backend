1.6.3 Lingua del titolo

Datatype: InformationService

CA element code: tit_langcoll_ME

Quicktip: Campo linked data. Specifica la lingua in cui è espresso il titolo della collezione, secondo codici standard ISO 639-2 o ISO 639-3. È possibile la ricerca per label della lingua (ad esempio "ital" per italiano, oppure per codice ISO,).

Campo Linked Data: Per i termini ammissibili, aggiornati al , clicca sul seguente link per eseguire la query in tempo reale: Esegui Query (attenzione: per query complesse occorre attendere diversi secondi prima della restituzione dei risultati).

SELECT DISTINCT ?item ?itemLabel ?itemDescription ?value ?lang
WHERE {
  {
   ?item wdt:P220 ?value;
     rdfs:label ?itemLabel.
 FILTER((LANG(?itemLabel)) = "it") 
  FILTER(REGEX(?itemLabel, "^(***PLACEHOLDER***|[^ ]+ ***PLACEHOLDER***)", "i"))
  }
UNION
 {
   ?item wdt:P220 ?value;
     rdfs:label ?itemLabel.
 FILTER((LANG(?itemLabel)) = "en") 
  FILTER(REGEX(?itemLabel, "^(***PLACEHOLDER***|[^ ]+ ***PLACEHOLDER***)", "i"))
  }
  UNION
  {
    ?item wdt:P220 ?value .
  FILTER (REGEX(?value, "***PLACEHOLDER***", "i"))  
    }  
    SERVICE wikibase:label { bd:serviceParam wikibase:language "it, en". }
BIND(LANG(?itemLabel) AS ?lang)
 }

1.6.4 Codice aggiuntivo lingua

Datatype: List

CA element code: record_res_add_lang_cocoll_ME

Quicktip: Inserisci qui i codici di lingua aggiuntivi previsti dallo standard SBN. Questi codici (alfanumerici a tre lettere, ISO 639‑2 integrato) permettono di specificare fino a tre lingue presenti nella risorsa, in ordine di prevalenza. Se sono presenti più di tre lingue, inserisci il codice della lingua predominante seguito da MUL (multilingue). Utilizza anche i codici convenzionali SBN per: MIS (lingue non codificate), UND (lingua non identificabile), ABS (risorsa senza contenuto linguistico).
