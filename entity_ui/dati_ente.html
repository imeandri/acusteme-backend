1.5.2 Corrispondenza con entità Wikidata

Datatype: InformationService

CA element code: wikidata_correlation_ME

Quicktip: Campo Linked Data per la selezione dell'entità corrispondente in Wikidata. L’associazione a una entità Wikidata (es. Q5582) consente l’integrazione con un vasto grafo semantico e l’eredità di proprietà esterne (es. VIAF, ISNI, BNF, LC). Si consiglia di validare sempre che l’entità selezionata sia corretta e aggiornata.

Campo Linked Data: Per i termini ammissibili, aggiornati al , clicca sul seguente link per eseguire la query in tempo reale: Esegui Query (attenzione: per query complesse occorre attendere diversi secondi prima della restituzione dei risultati).

select distinct ?item ?itemLabel ?itemDescription ?lang 
where {
   SERVICE wikibase:mwapi {
      bd:serviceParam wikibase:endpoint "www.wikidata.org";
                      wikibase:api "EntitySearch";
                      mwapi:search "***PLACEHOLDER***";
                      mwapi:language "it".
      ?item wikibase:apiOutputItem mwapi:item.
      ?num wikibase:apiOrdinal true.
  }
  SERVICE wikibase:label {
                   bd:serviceParam wikibase:language "[AUTO_LANGUAGE],it,en" .
                   }
  ?item (rdfs:label|skos:altLabel) ?itemLabel.
  OPTIONAL {
     ?item schema:description ?itemDescription.
      FILTER(LANGMATCHES(LANG(?itemDescription), "it") || LANGMATCHES(LANG(?itemDescription), "en"))
   }
  FILTER(LANGMATCHES(LANG(?itemLabel), "it") || LANGMATCHES(LANG(?itemLabel), "en"))
      FILTER(LANG(?itemLabel) = LANG(?itemDescription))
    BIND(LANG(?itemLabel) AS ?lang)
}

1.5.3 VIAF (da Wikidata)

Datatype: InformationService

CA element code: VIAF_IS

Quicktip: Campo Linked Data derivato (read-only o suggerito) che recupera l’identificatore VIAF (Virtual International Authority File) corrispondente all'entità, se presente in Wikidata. VIAF collega gli authority file di biblioteche nazionali e internazionali (LC, BNF, DNB, ecc.) in un’unica entità condivisa.

Campo Linked Data: Per i termini ammissibili, aggiornati al , clicca sul seguente link per eseguire la query in tempo reale: Esegui Query (attenzione: per query complesse occorre attendere diversi secondi prima della restituzione dei risultati).

SELECT distinct ?item ?itemLabel ?itemDescription ?value ?lang ?ViafLink
WHERE {
  SERVICE wikibase:label { bd:serviceParam wikibase:language "[AUTO_LANGUAGE],it,en". }
  
  ?item wdt:P214 ?value . 
  SERVICE wikibase:mwapi {
      bd:serviceParam wikibase:endpoint "www.wikidata.org";
                      wikibase:api "EntitySearch";
                      mwapi:search "***PLACEHOLDER***";
                      mwapi:language "it".
      ?item wikibase:apiOutputItem mwapi:item.
      ?num wikibase:apiOrdinal true.
  }
  ?item (rdfs:label|skos:altLabel) ?itemLabel.
  OPTIONAL {
     ?item schema:description ?itemDescription.
      FILTER(LANGMATCHES(LANG(?itemDescription), "it") || LANGMATCHES(LANG(?itemDescription), "en"))
   }
  FILTER(LANGMATCHES(LANG(?itemLabel), "it") || LANGMATCHES(LANG(?itemLabel), "en"))
      FILTER(LANG(?itemLabel) = LANG(?itemDescription))
    BIND(LANG(?itemLabel) AS ?lang)
   BIND(
  CONCAT("VIAF: ", "https://viaf.org/viaf/", ?value) AS ?ViafLink
)
  
 }
order by ?item
