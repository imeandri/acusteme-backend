Quicktip: Questo container permette l'inserimento di dati demografici e linguistici relativi a gruppi demografici documentati dalla risorsa.

1.1.1 Gruppo demografico e/o appartenenza religiosa

Datatype: InformationService

CA element code: ethnic_group_dem2_ME

Quicktip: Campo Linked Data. Indicare una o più macro-categorie demografiche relative alla risorsa etnomusicologica. Es. rom, sinti, curdi iraniani, rumoristi, scalpellini.

Campo Linked Data: Per i termini ammissibili, aggiornati al , clicca sul seguente link per eseguire la query in tempo reale: Esegui Query (attenzione: per query complesse occorre attendere diversi secondi prima della restituzione dei risultati).

SELECT distinct ?item ?itemLabel ?itemDescription ?lang
WHERE {
  SERVICE wikibase:label { bd:serviceParam wikibase:language "[AUTO_LANGUAGE],it, en". }
 { ?item wdt:P31 ?ist .
  VALUES ?ist {wd:Q11197007 wd:Q41710 wd:Q4392985 wd:Q217438}
    MINUS {?item wdt:P31 wd:Q2472587}
    MINUS {?item wdt:P31 wd:Q6266}
    MINUS {?item wdt:P31 wd:Q231002}
    MINUS {?item wdt:P31 wd:Q33829}
    MINUS {?item wdt:P31 wd:Q81058955}
 ?item (rdfs:label|skos:altLabel) ?itemLabel.
    FILTER(REGEX(?itemLabel, "^(***PLACEHOLDER***|[^ ]+ ***PLACEHOLDER***)", "i"))
  ?item schema:description ?itemDescription.
    }
  UNION
  {
  ?item wdt:P4946 ?id . 
  ?item (rdfs:label|skos:altLabel) ?itemLabel.
    FILTER(REGEX(?itemLabel, "^(***PLACEHOLDER***|[^ ]+ ***PLACEHOLDER***)", "i"))
  ?item schema:description ?itemDescription.
    }
  FILTER(LANGMATCHES(LANG(?itemLabel), "it") || LANGMATCHES(LANG(?itemLabel), "en"))
  FILTER(LANG(?itemLabel) = LANG(?itemDescription))
    BIND(LANG(?itemLabel) AS ?lang)
}
ORDER BY (?item)

1.1.3 Famiglie e gruppi linguistici (ISO639-5)

Datatype: InformationService

CA element code: ISO639_5_3

Quicktip: Campo Linked Data: cercare utilizzando il codice lingua ISO relativo alla famiglia o gruppo linguistico (ad es. 'ber' per "Berber languages"; 'sgn' per "Sign languages"). Per un elenco completo dei codici ISO relativi alle famiglie linguistiche fare riferimento alla documentazione.

Campo Linked Data: Per i termini ammissibili, aggiornati al , clicca sul seguente link per eseguire la query in tempo reale: Esegui Query (attenzione: per query complesse occorre attendere diversi secondi prima della restituzione dei risultati).

SELECT DISTINCT ?item ?itemLabel ?itemDescription ?value  
WHERE {
  {
   ?item wdt:P1798 ?value;
     rdfs:label ?itemLabel.
 FILTER((LANG(?itemLabel)) = "it") 
  FILTER(REGEX(?itemLabel, "^(***PLACEHOLDER***|[^ ]+ ***PLACEHOLDER***)", "i"))
  }
UNION
 {
   ?item wdt:P1798 ?value;
     rdfs:label ?itemLabel.
 FILTER((LANG(?itemLabel)) = "en") 
  FILTER(REGEX(?itemLabel, "^(***PLACEHOLDER***|[^ ]+ ***PLACEHOLDER***)", "i"))

  UNION
  {
    ?item wdt:P1798 ?value .
  FILTER (REGEX(?value, "***PLACEHOLDER***", "i"))
