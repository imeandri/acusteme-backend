1.1 Identificatore standard

Datatype: Container

CA element code: standard_identifier_CN

Ripetibilità: Sì

Quicktip: Contenitore strutturato per l’inserimento degli identificatori standard associati a una manifestazione, secondo le normative ISBD (es. ISBN, ISMN, ISSN, SBN, ecc.). Se pertinente, inserire anche eventuali note e titolo chiave collegato all’ISSN.

1.1.1 Tipologia identificatore

Datatype: List

CA element code: standard_identifier_ME

Quicktip: Selezionare dal vocabolario controllato la tipologia dell’identificatore standard (es. ISBN, ISMN, ISSN, SBN, ecc.).
