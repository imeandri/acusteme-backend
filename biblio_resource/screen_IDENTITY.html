1.5 Tipo di risorsa (etnomusicologia, musica per film)

Datatype: InformationService

CA element code: typeOfEthnoFMRes_ME

Quicktip: Campo Linked Data. Indicare il tipo di risorsa (e.g. “Cue Sheet”, “Documento sonoro di registrazione sul campo”). Inserire almeno tre lettere per la ricerca Linked Data e scegliere tra le alternative proposte. Per l'elenco dei termini ammessi consultare la documentazione (i).

Campo Linked Data: Per i termini ammissibili, aggiornati al , clicca sul seguente link per eseguire la query in tempo reale: Esegui Query (attenzione: per query complesse occorre attendere diversi secondi prima della restituzione dei risultati).

SELECT DISTINCT ?item ?itemLabel ?itemDescription ?lang
WHERE {
    ?item wdt:P5008 wd:Q120325256.
    ?item wdt:P2579 ?discipline .
    VALUES ?discipline {wd:Q208365 wd:Q164204}
    ?item wdt:P279 wd:Q49848 .
    ?item(rdfs:label|skos:altLabel) ?itemLabel.
    FILTER(REGEX(?itemLabel, "^(***PLACEHOLDER***|***PLACEHOLDER*** |[^ ]+ ***PLACEHOLDER***)", "i"))
    FILTER(LANGMATCHES(LANG(?itemLabel), "it") || LANGMATCHES(LANG(?itemLabel), "en"))
    ?item schema:description ?itemDescription.
    FILTER(LANGMATCHES(LANG(?itemDescription), "it") || LANGMATCHES(LANG(?itemDescription), "en"))
    FILTER(LANG(?itemLabel) = LANG(?itemDescription))
   
  BIND(LANG(?itemLabel) AS ?lang)
  SERVICE wikibase:label { bd:serviceParam wikibase:language "[AUTO_LANGUAGE], en, it". }  

} 
order by ?item

1.7 Lingua/e del contenuto (ISO 639-3)

Datatype: InformationService

CA element code: cont_lang_ME

Ripetibilità: Sì

Quicktip: Campo Linked Data. Indicare la lingua o le lingue principali del contenuto. Avviare la ricerca Linked Data inserendo almeno tre lettere utilizzando il codice lingua ISO (ad es. 'kmr' per il curdo settentrionale) o il nome lingua ('ital' per 'italiano' o 'lingua dei segni italiana'). Registrare tutte le lingue del contenuto, o registrare almeno la lingua principale e indicare che la risorsa è multilingue nel campo successivo. Elemento ripetibile.

Campo Linked Data: Per i termini ammissibili, aggiornati al , clicca sul seguente link per eseguire la query in tempo reale: Esegui Query (attenzione: per query complesse occorre attendere diversi secondi prima della restituzione dei risultati).

SELECT DISTINCT ?item ?itemLabel ?itemDescription ?value ?lang
WHERE {
  {
   ?item wdt:P220 ?value;
     rdfs:label ?itemLabel.
 FILTER((LANG(?itemLabel)) = "it") 
  FILTER(REGEX(?itemLabel, "^(***PLACEHOLDER***|[^ ]+ ***PLACEHOLDER***)", "i"))
  }
UNION
 {
   ?item wdt:P220 ?value;
     rdfs:label ?itemLabel.
 FILTER((LANG(?itemLabel)) = "en") 
  FILTER(REGEX(?itemLabel, "^(***PLACEHOLDER***|[^ ]+ ***PLACEHOLDER***)", "i"))
  }
  UNION
  {
    ?item wdt:P220 ?value .
  FILTER (REGEX(?value, "***PLACEHOLDER***", "i"))  
    }  
    SERVICE wikibase:label { bd:serviceParam wikibase:language "it, en". }
BIND(LANG(?itemLabel) AS ?lang)
 }
