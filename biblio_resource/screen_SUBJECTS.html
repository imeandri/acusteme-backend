1.2 Thesaurus etnografico dell’AFS

Datatype: InformationService

CA element code: ethno_thesaurus_IS

Ripetibilità: Sì

Quicktip: Campo Linked Data. Il Thesaurus etnografico dell'American Folklore Society è un vocabolario online ricercabile con informazioni sull'etnomusicologia, l'antropologia culturale e i campi correlati. Per il corretto uso dei termini fare riferimento alla documentazione.

Campo Linked Data: Per i termini ammissibili, aggiornati al , clicca sul seguente link per eseguire la query in tempo reale: Esegui Query (attenzione: per query complesse occorre attendere diversi secondi prima della restituzione dei risultati).

SELECT DISTINCT ?item ?itemLabel ?itemDescription ?LOCLink ?lang
WHERE {
  
    ?item wdt:P8540 ?id.
    ?item(rdfs:label|skos:altLabel) ?itemLabel.
    FILTER(REGEX(?itemLabel, "^(***PLACEHOLDER***|[^ ]+ ***PLACEHOLDER***)", "i"))
    FILTER(LANGMATCHES(LANG(?itemLabel), "it") || LANGMATCHES(LANG(?itemLabel), "en"))
    ?item schema:description ?itemDescription.
    FILTER(LANGMATCHES(LANG(?itemDescription), "it") || LANGMATCHES(LANG(?itemDescription), "en"))
    FILTER(LANG(?itemLabel) = LANG(?itemDescription))
   
  BIND(LANG(?itemLabel) AS ?lang)
  SERVICE wikibase:label { bd:serviceParam wikibase:language "[AUTO_LANGUAGE], en, it". } 
   BIND(
  CONCAT(
    "LOC Vocabulary: ",  
    IF(BOUND(?id), " ", ""), 
    CONCAT("https://id.loc.gov/vocabulary/ethnographicTerms/", ?id)
  ) AS ?LOCLink
)
} 
order by ?item

1.4 Thesaurus dell'UNESCO

Datatype: InformationService

CA element code: unesco_thesaurus_IS

Ripetibilità: Sì

Quicktip: Campo Linked Data. Il Thesaurus dell'UNESCO è un elenco controllato e strutturato di termini utilizzati per l'analisi e il recupero di documenti e pubblicazioni nei settori dell'istruzione, della cultura, delle scienze naturali, delle scienze sociali e umane, della comunicazione e dell'informazione.

Campo Linked Data: Per i termini ammissibili, aggiornati al , clicca sul seguente link per eseguire la query in tempo reale: Esegui Query (attenzione: per query complesse occorre attendere diversi secondi prima della restituzione dei risultati).

SELECT DISTINCT ?item ?itemLabel ?itemDescription ?lang ?id ?Unesco_id 
WHERE {
  
  ?item wdt:P3916 ?id.

  ?item (rdfs:label|skos:altLabel) ?itemLabel.

  FILTER(CONTAINS(LCASE(?itemLabel), "***PLACEHOLDER***"))

  FILTER(LANGMATCHES(LANG(?itemLabel), "it") || LANGMATCHES(LANG(?itemLabel), "en"))

  OPTIONAL {
    ?item schema:description ?itemDescription.
    FILTER(LANGMATCHES(LANG(?itemDescription), "it") || LANGMATCHES(LANG(?itemDescription), "en"))
  }

  BIND(LANG(?itemLabel) AS ?lang)
  SERVICE wikibase:label { bd:serviceParam wikibase:language "en,it". }

  BIND(CONCAT("Unesco Thesaurus: ", "https://vocabularies.unesco.org/thesaurus/", ?id) AS ?Unesco_id)

} 
ORDER BY ?item
LIMIT 50

La selezione può includere: guerre, insurrezioni, migrazioni forzate rivoluzioni, regimi politici, transizioni democratiche momenti chiave della storia locale, nazionale o globale Cercare utilizzando parole chiave (almeno tre lettere) o codici URI, basati su risorse esterne (Wikidata), L’uso del campo consente di contestualizzare la risorsa in una prospettiva storico-politica, facilitando l’interoperabilità con archivi, mappe e cronologie semantiche.
