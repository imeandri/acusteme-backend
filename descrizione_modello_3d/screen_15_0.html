1.4 Tipologia di scansione

Datatype: List

CA element code: scan_type_ME

Quicktip: Descrive la tecnologia o il metodo utilizzato per acquisire la rappresentazione 3D. Esempi: “Laser scanning”, “Fotogrammetria”, “Structured light”, “CT scan”. Utile per valutare l’accuratezza e la natura dei dati acquisiti.

1.5 Risoluzione

Datatype: List

CA element code: 3D_resolution_ME

Quicktip: Indica il livello di dettaglio della rappresentazione 3D, espresso in punti per unità (es. punti/mm) o densità poligonale. Valore utile per confrontare la precisione tra diverse acquisizioni o per scopi di restauro, stampa o analisi.

Esempi d'uso: {to be completed}

1.6 Formato

Datatype: List

CA element code: 3D_format_ME

Quicktip: Specifica il formato del file 3D. Esempi comuni: .obj, .stl, .ply, .glb, .fbx, .3mf. Fondamentale per garantire la compatibilità con software di visualizzazione, modellazione o conservazione.

Esempi d'uso: {to be completed}

1.7 Scala

Datatype: List

CA element code: 3D_scale_ME

Quicktip: Indica il rapporto tra le dimensioni del modello 3D e l’oggetto reale. Esempio: 1:1 per modello in scala reale, 1:10 per modello ridotto. Importante per comprendere l’accuratezza della rappresentazione rispetto all’originale.
