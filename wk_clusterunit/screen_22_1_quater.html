1.1 Data di produzione

Datatype: DateRange

CA element code: filmcomp_date_ME

Quicktip: Specifica la data ufficiale di produzione del film. Può essere nazionale o internazionale. Campo 'daterange', inserire date singole. Esempi: '30 Agosto 1746', '1900', '7 Giugno 2007', 'Giugno 2016'. Collective access supporta anche date-range ed espressioni di date incerte nei campi date-range, ma è consigliabile utilizzare le normative bibliografiche, che esprimono intervalli incerti attraverso il metodo DATA 1-DATA2 (v. documentazione).

1.4.2 Film (IMDB da Wikidata)

Datatype: InformationService

CA element code: film_imdb_ind_ME

Quicktip: Campo Linked Data che consente il collegamento al record Wikidata del film, sfruttando il collegamento via ID IMDb. Inserire almeno tre lettere del titolo per attivare il motore di ricerca.

Campo Linked Data: Per i termini ammissibili, aggiornati al , clicca sul seguente link per eseguire la query in tempo reale: Esegui Query (attenzione: per query complesse occorre attendere diversi secondi prima della restituzione dei risultati).

SELECT DISTINCT ?item ?itemLabel ?itemDescription ?lang ?imdbID ?IMDbLink
WHERE {
 
  ?item wdt:P345 ?imdbID;
        rdfs:label ?itemLabel.
 
  FILTER((LANG(?itemLabel)) = "en")
  FILTER(REGEX(?itemLabel, "^(***PLACEHOLDER***|[^ ]+ ***PLACEHOLDER***)", "i"))

 
  ?item wdt:P31 wd:Q11424.

  
  OPTIONAL {
    ?item schema:description ?itemDescription.
    FILTER(LANGMATCHES(LANG(?itemDescription), "en"))
  }
   BIND(LANG(?itemLabel) AS ?lang)

BIND(
  CONCAT("IMDb: ", "https://www.imdb.com/it/title/", ?imdbID) AS ?IMDbLink)
}
ORDER BY ?item
LIMIT 100
