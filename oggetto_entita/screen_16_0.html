1.3.4 Denominazione locale del performer/artista o dell'ensemble

Datatype: InformationService

CA element code: performed_localden_ME

Quicktip: Campo Linked Data. Inserire la denominazione locale con cui il performer è conosciuto nel proprio contesto culturale o comunità, ad es. in dialetto, lingua minoritaria, o con titolo/qualifica tradizionale.

Campo Linked Data: Per i termini ammissibili, aggiornati al , clicca sul seguente link per eseguire la query in tempo reale: Esegui Query (attenzione: per query complesse occorre attendere diversi secondi prima della restituzione dei risultati).

SELECT ?item ?itemLabel ?itemDescription ?lang WHERE {
  SERVICE wikibase:mwapi {
    bd:serviceParam wikibase:endpoint "www.wikidata.org";
                    wikibase:api "EntitySearch";
                    mwapi:search "***PLACEHOLDER***";
                    mwapi:language "it".
    ?item wikibase:apiOutputItem mwapi:item.
    ?num wikibase:apiOrdinal true.
  }

  OPTIONAL { ?item rdfs:label ?labIt FILTER(LANG(?labIt)="it") }
  OPTIONAL { ?item rdfs:label ?labEn FILTER(LANG(?labEn)="en") }
  BIND(COALESCE(?labIt, ?labEn) AS ?itemLabel)
  BIND(LANG(?itemLabel) AS ?lang)

  OPTIONAL { ?item schema:description ?dIt FILTER(LANG(?dIt)="it") }
  OPTIONAL { ?item schema:description ?dEn FILTER(LANG(?dEn)="en") }
  BIND(COALESCE(?dIt, ?dEn, "") AS ?itemDescription)
}
ORDER BY ?num
LIMIT 50
