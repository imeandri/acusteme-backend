1.4.1 Famiglie e gruppi linguistici (ISO639-5)

Datatype: InformationService

CA element code: ISO639_5_2

Quicktip: Campo Linked Data: cercare utilizzando il codice lingua ISO relativo alla famiglia o gruppo linguistico (ad es. 'ber' per "Berber languages"; 'sgn' per "Sign languages"). Per un elenco completo dei codici ISO relativi alle famiglie linguistiche fare riferimento alla documentazione.

Campo Linked Data: Per i termini ammissibili, aggiornati al , clicca sul seguente link per eseguire la query in tempo reale: Esegui Query (attenzione: per query complesse occorre attendere diversi secondi prima della restituzione dei risultati).

SELECT DISTINCT ?item ?itemLabel ?itemDescription ?value  
WHERE {
  {
   ?item wdt:P1798 ?value;
     rdfs:label ?itemLabel.
 FILTER((LANG(?itemLabel)) = "it") 
  FILTER(REGEX(?itemLabel, "^(***PLACEHOLDER***|[^ ]+ ***PLACEHOLDER***)", "i"))
  }
UNION
 {
   ?item wdt:P1798 ?value;
     rdfs:label ?itemLabel.
 FILTER((LANG(?itemLabel)) = "en") 
  FILTER(REGEX(?itemLabel, "^(***PLACEHOLDER***|[^ ]+ ***PLACEHOLDER***)", "i"))
  }
  UNION
  {
    ?item wdt:P1798 ?value .
  FILTER (REGEX(?value, "***PLACEHOLDER***", "i"))  
    }  
    SERVICE wikibase:label { bd:serviceParam wikibase:language "it, en". }

 }

1.5.2 Denominazione locale (lod)

Datatype: InformationService

CA element code: denominazione_locale_strum2

Quicktip: Campo Linked Data. Inserire almeno tre lettere e scegliere tra le alternative proposte. Per un elenco completo delle denominazioni locali disponibili e per la richiesta di inserimento di nuove denominazioni locali fare riferimento alla documentazione.

Campo Linked Data: Per i termini ammissibili, aggiornati al , clicca sul seguente link per eseguire la query in tempo reale: Esegui Query (attenzione: per query complesse occorre attendere diversi secondi prima della restituzione dei risultati).

SELECT DISTINCT ?displayValue ?item ?itemDescription ?lang
WHERE {
  {
    ?item wdt:P31 ?ist.
    VALUES ?ist {wd:Q1254773 wd:Q110295396}
    ?item(rdfs:label|skos:altLabel) ?itemLabel.
    FILTER(REGEX(?itemLabel, "^(***PLACEHOLDER***|[^ ]+ ***PLACEHOLDER***)", "i"))
  }
  UNION
  {
  ?item wdt:P31 ?ist.
    VALUES ?ist {wd:Q1254773 wd:Q110295396}
    ?item wdt:P1705 ?name .
    FILTER(REGEX(?name, "^(***PLACEHOLDER***|[^ ]+ ***PLACEHOLDER***)", "i"))
  }
  UNION
  {
    ?item wdt:P279* wd:Q34379 .
    ?item(rdfs:label|skos:altLabel) ?itemLabel.
    FILTER(REGEX(?itemLabel, "^(***PLACEHOLDER***|[^ ]+ ***PLACEHOLDER***)", "i"))
    }
  UNION
  {
    ?item wdt:P279* wd:Q34379 .
    ?item wdt:P1705 ?name .
    FILTER(REGEX(?name, "^(***PLACEHOLDER***|[^ ]+ ***PLACEHOLDER***)", "i"))
    }

  #FILTER(LANGMATCHES(LANG(COALESCE(?itemLabel, ?name)), "it") || LANGMATCHES(LANG(COALESCE(?itemLabel, ?name)), "en"))
  BIND(LANG(COALESCE(?itemLabel, ?name)) AS ?lang)
  BIND(COALESCE(?itemLabel, ?name) AS ?displayValue)
  OPTIONAL {
    ?item schema:description ?itemDescription.
    FILTER(LANG(?itemDescription) = ?lang)
  }
  SERVICE wikibase:label { bd:serviceParam wikibase:language "[AUTO_LANGUAGE], en, it". }  

}
order by ?item
