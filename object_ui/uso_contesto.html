1.2.1 data di inizio

CA element code: inizio_uso

Quicktip: Campo 'daterange', inserire date singole o, per date incerte, intervalli temporali. Esempi: '30 Agosto 1746', '1900', '7 Giugno 2007 alle 15:31:48', 'Giugno 2016'. Per date incerte, utilizzare il seguenti formati: 'circa 1955', '1955 ~ 3y' (1955 più o meno 3 anni), 'circa 1950 - 1956', '10 Giugno 1955 ~ 10d' (10 giugno 1955 più o meno 10 giorni), '850 BCE', '40 CE', '6/10/????' oppure '06/10/?' (data con indicazione di giorno e mese, ma senza anno), '6/????' (data con indicazione di mese, ma senza anno). È possibile utilizzare anche i seguenti valori, che verranno poi trasformati dal sistema in un range di date effettive: 'autunno 2003' (indicazione di stagione), '20 Q3' (terzo quarto del ventesimo secolo).

1.3 Luogo d'uso

Datatype: Container

CA element code: use_place_CN

Ripetibilità: Sì

Quicktip: inserire, se noti e pertinenti, uno o più luoghi d'uso dello strumento. Si utilizza questo campo per georeferenziazioni corripondenti a divisioni amministrative (prevalentemente città, province, regioni)., mentre si utilizza il successivo campo Coordinate/areale per localizzazioni di singole o multiple coordinate geografiche o areali.

1.3.1 Geonames

Datatype: GeoNames

CA element code: geonames_uso

Quicktip: Campo Linked data (database GeoNames). Inserire il nome di una città, in qualsiasi lingua (Es. Riva presso Chieri, Sulaymaniyah o سلێمانی , Pechino o 北京市, Teheran o تهران, ) e scegliere tra le proposte della base di dati. Per ricerche più precise possono essere utilizzati i codici Geonames: ADM1 (first-order administrative division (per l'Italia, Regione – la ricerca ADM1 Italia restituisce tutte le regioni italiane – per gli Stati Uniti: gli Stati – la ricerca ); ADM2 (second-order administrative division, es ADM2 Torino la provincia di Torino), ADM3 Torino (la città di Torino). Le coordinate puntano al centro geografico dell'area individuata. Per i luoghi storici è possibile utilizzare il suffisso h, ad esempio: ADM2H, Carbonia-Iglesias (ex provincia italiana); PCLH (historical political entity, es. PCLH, Repubblica Democratica Tedesca; L'uso dei codici di localizzazione consente la definizione di molteplici aree (di geografia fisica e territori informali, ad esempio STNB, scientific research base (STNB Stazione McMurdo, Antartide), TERR, territory (TERR Sahara Occidentale); GYSR, geyser, GYSR Géiser del Tatio, Cile, SA. Per altri codici conultare la documentazione.

1.7 Evento rituale/occasione

Datatype: InformationService

CA element code: ritual_event_ME

Ripetibilità: Sì

Quicktip: Campo Linked Data: indicare, se pertinente, uno o più eventi rituali o occasioni relativi alla risorsa etnomusicologica. Cercare inserendo almeno tre lettere per attivare la ricerca, ad esempio 'car' per 'Carnevale di Ivrea'; 'Now' o 'Naw' (o qualsiasi grafia alternativa con cui il nome è registrato su wikidata, es. central kurdish 'نه ورۆز') per 'Nowruz'.

Campo Linked Data: Per i termini ammissibili, aggiornati al , clicca sul seguente link per eseguire la query in tempo reale: Esegui Query (attenzione: per query complesse occorre attendere diversi secondi prima della restituzione dei risultati).

SELECT DISTINCT
  ?item
  ?keyLabel
  ?keyDesc
  ?lang
  ?layer
  ?placeLabel
WHERE {
  # Gate di curatela (solo focus)
  ?item wdt:P5008 wd:Q120325256 .

  # Match IT/EN su label o altLabel
  {
    ?item (rdfs:label|skos:altLabel) ?lab .
    FILTER(LANG(?lab) = "it")
    FILTER(REGEX(STR(?lab), "^(***PLACEHOLDER***|[^ ]+ ***PLACEHOLDER***)", "i"))
    BIND("it" AS ?lang)
    BIND(?lab AS ?rawKeyLabel)
  }
  UNION
  {
    ?item (rdfs:label|skos:altLabel) ?lab .
    FILTER(LANG(?lab) = "en")
    FILTER(REGEX(STR(?lab), "^(***PLACEHOLDER***|[^ ]+ ***PLACEHOLDER***)", "i"))
    BIND("en" AS ?lang)
    BIND(?lab AS ?rawKeyLabel)
  }

  # Description (fallback incrociato IT-EN)
  OPTIONAL { ?item schema:description ?descIt FILTER(LANG(?descIt)="it") }
  OPTIONAL { ?item schema:description ?descEn FILTER(LANG(?descEn)="en") }
  BIND(
    IF(?lang="it", COALESCE(?descIt, ?descEn),
                 COALESCE(?descEn, ?descIt)
    ) AS ?rawKeyDesc
  )

  # Luogo opzionale (location, se c'è)
  OPTIONAL {
    ?item wdt:P276 ?place .
    OPTIONAL { ?place rdfs:label ?pIt FILTER(LANG(?pIt)="it") }
    OPTIONAL { ?place rdfs:label ?pEn FILTER(LANG(?pEn)="en") }
    BIND(STR(COALESCE(?pIt, ?pEn, "")) AS ?placeLabel)
  }

  # Layer fisso
  BIND("ritual_occasion" AS ?layer)

  # Stringhe plain per CA
  BIND(STR(?rawKeyLabel) AS ?keyLabel)
  BIND(STR(COALESCE(?rawKeyDesc, "")) AS ?keyDesc)
}
ORDER BY ?lang ?keyLabel
LIMIT 200
