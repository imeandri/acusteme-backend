# ACUSTEME Installation Profile

This directory contains the ACUSTEME CollectiveAccess installation profile.

Files:

- `ACUSTEME_profile.xml`: installation profile, currently including Italian, English, and Romanian locales as present in the source profile. The current public review focuses primarily on Italian and English labels/descriptions.
- `profile.xsd`: schema used for XML validation.

This profile is a first public draft. It reflects the current ACUSTEME backend configuration and still requires review before reuse in other institutional contexts.

Known limitations include incomplete SPARQL generalization, automatic English translation requiring review, and controlled lists that are not yet fully normalized.
