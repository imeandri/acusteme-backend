1.1 General description

Datatype: Text

CA element code: general_description_ME

Quicktip: Report in this field information about the morphology of the musical instruments and to any decorations. If there are several similar instruments in the same collection, it is in the field devoted to description (in addition to the field devoted to measurements) that the differences between the specimens should be specified.
