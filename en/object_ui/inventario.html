1.3.1 Inventory type

Datatype: List

CA element code: tipo_altro_inventario

Quicktip: Specifies the type of other inventory to which the object belongs. Use this option to distinguish between: Previous inventory: historical number assigned in an old system or by another entity; Special inventory: number assigned as part of thematic, extraordinary, or temporary cataloging. Select the most relevant item with respect to the context of origin or use of the inventory number.

Datatype: DateRange

CA element code: altro_inventario_inizio

Quicktip: Daterange' field, enter single dates or, for uncertain dates, time intervals. Examples: 'August 30, 1746', '1900', 'June 7, 2007 at 15:31:48', 'June 2016'. For uncertain dates, use the following formats: 'about 1955', '1955 ~ 3y' (1955 plus or minus 3 years), 'about 1950 - 1956', 'June 10, 1955 ~ 10d' (June 10, 1955 plus or minus 10 days), '850 BCE', '40 CE', '6/10/????' or '06/10/?' (date with day and month indicated, but no year), '6/????' (date with month indicated, but no year). The following values can also be used, which will then be transformed by the system into a range of actual dates: 'autumn 2003' (season indication), '20 Q3' (third quarter of the twentieth century).

1.4.1 RFID identification code

Datatype: Text

CA element code: codice_identificativo_rfid

Quicktip: Unique code associated with the RFID (Radio-Frequency Identification) tag physically applied to the resource. Used for automatic identification and tracking in inventory, handling, loan or security systems. The code can be numeric or alphanumeric and is usually read by an RFID reader without the need for visual contact. Verifies that the value entered is consistent with the tag actually applied to the item or medium.
