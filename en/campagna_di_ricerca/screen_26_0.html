1.5.1 Demographic group

Datatype: InformationService

CA element code: ethnic_group_dem2_ME

Quicktip: Linked Data field. Indicate one or more demographic macro-categories related to the ethnomusicological resource. E.g., Roma, Sinti, Iranian Kurds, foleys, stonemasons.

Linked Data field: for the allowed terms, updated on , click the following link to run the query in real time: Run Query (for complex queries, wait several seconds before results are returned).

SELECT distinct ?item ?itemLabel ?itemDescription ?lang
WHERE {
  SERVICE wikibase:label { bd:serviceParam wikibase:language "[AUTO_LANGUAGE],it, en". }
 { ?item wdt:P31 ?ist .
  VALUES ?ist {wd:Q11197007 wd:Q41710 wd:Q4392985 wd:Q217438}
    MINUS {?item wdt:P31 wd:Q2472587}
    MINUS {?item wdt:P31 wd:Q6266}
    MINUS {?item wdt:P31 wd:Q231002}
    MINUS {?item wdt:P31 wd:Q33829}
    MINUS {?item wdt:P31 wd:Q81058955}
 ?item (rdfs:label|skos:altLabel) ?itemLabel.
    FILTER(REGEX(?itemLabel, "^(***PLACEHOLDER***|[^ ]+ ***PLACEHOLDER***)", "i"))
  ?item schema:description ?itemDescription.
    }
  UNION
  {
  ?item wdt:P4946 ?id . 
  ?item (rdfs:label|skos:altLabel) ?itemLabel.
    FILTER(REGEX(?itemLabel, "^(***PLACEHOLDER***|[^ ]+ ***PLACEHOLDER***)", "i"))
  ?item schema:description ?itemDescription.
    }
  FILTER(LANGMATCHES(LANG(?itemLabel), "it") || LANGMATCHES(LANG(?itemLabel), "en"))
  FILTER(LANG(?itemLabel) = LANG(?itemDescription))
    BIND(LANG(?itemLabel) AS ?lang)
}
ORDER BY (?item)

1.5.3 Language families and groups (ISO639-5)

Datatype: InformationService

CA element code: ISO639_5_3

Quicktip: Linked Data field: search using the ISO language code related to the language family or group (e.g., 'ber' for "Berber languages"; 'sgn' for "Sign languages"). For a complete list of ISO codes related to language families refer to the documentation.

Linked Data field: for the allowed terms, updated on , click the following link to run the query in real time: Run Query (for complex queries, wait several seconds before results are returned).

SELECT DISTINCT ?item ?itemLabel ?itemDescription ?value  
WHERE {
  {
   ?item wdt:P1798 ?value;
     rdfs:label ?itemLabel.
 FILTER((LANG(?itemLabel)) = "it") 
  FILTER(REGEX(?itemLabel, "^(***PLACEHOLDER***|[^ ]+ ***PLACEHOLDER***)", "i"))
  }
UNION
 {
   ?item wdt:P1798 ?value;
     rdfs:label ?itemLabel.
 FILTER((LANG(?itemLabel)) = "en") 
  FILTER(REGEX(?itemLabel, "^(***PLACEHOLDER***|[^ ]+ ***PLACEHOLDER***)", "i"))

  UNION
  {
    ?item wdt:P1798 ?value .
  FILTER (REGEX(?value, "***PLACEHOLDER***", "i"))

    SERVICE wikibase:label { bd:serviceParam wikibase:language "it, en". }

 }
