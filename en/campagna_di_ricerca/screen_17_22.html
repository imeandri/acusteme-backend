1.2 AFS ethnographic thesaurus

Datatype: InformationService

CA element code: ethno_thesaurus_IS

Repeatability: Yes

Quicktip: Linked Data field. The AFS (American Folklore Society) Ethnographic Thesaurus is a searchable online vocabulary with information on ethnomusicology, cultural anthropology, and related fields. Refer to the documentation for correct usage of terms.

Linked Data field: for the allowed terms, updated on , click the following link to run the query in real time: Run Query (for complex queries, wait several seconds before results are returned).

SELECT DISTINCT ?item ?itemLabel ?itemDescription ?LOCLink ?lang
WHERE {
  
    ?item wdt:P8540 ?id.
    ?item(rdfs:label|skos:altLabel) ?itemLabel.
    FILTER(REGEX(?itemLabel, "^(***PLACEHOLDER***|[^ ]+ ***PLACEHOLDER***)", "i"))
    FILTER(LANGMATCHES(LANG(?itemLabel), "it") || LANGMATCHES(LANG(?itemLabel), "en"))
    ?item schema:description ?itemDescription.
    FILTER(LANGMATCHES(LANG(?itemDescription), "it") || LANGMATCHES(LANG(?itemDescription), "en"))
    FILTER(LANG(?itemLabel) = LANG(?itemDescription))
   
  BIND(LANG(?itemLabel) AS ?lang)
  SERVICE wikibase:label { bd:serviceParam wikibase:language "[AUTO_LANGUAGE], en, it". } 
   BIND(
  CONCAT(
    "LOC Vocabulary: ",  
    IF(BOUND(?id), " ", ""), 
    CONCAT("https://id.loc.gov/vocabulary/ethnographicTerms/", ?id)
  ) AS ?LOCLink
)
} 
order by ?item

1.3 UNESCO Thesaurus

Datatype: InformationService

CA element code: unesco_thesaurus_IS

Repeatability: Yes

Quicktip: Linked Data field. The UNESCO Thesaurus is a controlled and structured list of terms used for the analysis and retrieval of documents and publications in the fields of education, culture, natural sciences, social sciences and humanities, communication and information.

Linked Data field: for the allowed terms, updated on , click the following link to run the query in real time: Run Query (for complex queries, wait several seconds before results are returned).

SELECT DISTINCT ?item ?itemLabel ?itemDescription ?lang ?id ?Unesco_id 
WHERE {
  
  ?item wdt:P3916 ?id.

  ?item (rdfs:label|skos:altLabel) ?itemLabel.

  FILTER(CONTAINS(LCASE(?itemLabel), "***PLACEHOLDER***"))

  FILTER(LANGMATCHES(LANG(?itemLabel), "it") || LANGMATCHES(LANG(?itemLabel), "en"))

  OPTIONAL {
    ?item schema:description ?itemDescription.
    FILTER(LANGMATCHES(LANG(?itemDescription), "it") || LANGMATCHES(LANG(?itemDescription), "en"))
  }

  BIND(LANG(?itemLabel) AS ?lang)
  SERVICE wikibase:label { bd:serviceParam wikibase:language "en,it". }

  BIND(CONCAT("Unesco Thesaurus: ", "https://vocabularies.unesco.org/thesaurus/", ?id) AS ?Unesco_id)

} 
ORDER BY ?item
LIMIT 50

Datatype: InformationService
