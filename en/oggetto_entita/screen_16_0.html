CA element code: fict_INT

Quicktip: Check the field if it is known that responsibility (e.g., author, performer, composer, copyist, etc.) is attributed fictitiously or conventionally. This also applies to collectively used pseudonyms or invented names assigned by tradition. Typical use: Traditional attributions not supported by firm documentary evidence. Literary or mythical figures with whom a work is conventionally associated. Known pseudonyms used by cataloguing convention (e.g., "Onkel Hans" in children's collections).

1.3.4 Local denomination of the performer/artist or ensemble

Datatype: InformationService

CA element code: performed_localden_ME

Quicktip: Linked Data field. Enter the local name by which the performer is known in his/her cultural context or community, e.g., dialect, minority language, or traditional title/qualification.

Linked Data field: for the allowed terms, updated on , click the following link to run the query in real time: Run Query (for complex queries, wait several seconds before results are returned).

SELECT ?item ?itemLabel ?itemDescription ?lang WHERE {
  SERVICE wikibase:mwapi {
    bd:serviceParam wikibase:endpoint "www.wikidata.org";
                    wikibase:api "EntitySearch";
                    mwapi:search "***PLACEHOLDER***";
                    mwapi:language "it".
    ?item wikibase:apiOutputItem mwapi:item.
    ?num wikibase:apiOrdinal true.
  }

  OPTIONAL { ?item rdfs:label ?labIt FILTER(LANG(?labIt)="it") }
  OPTIONAL { ?item rdfs:label ?labEn FILTER(LANG(?labEn)="en") }
  BIND(COALESCE(?labIt, ?labEn) AS ?itemLabel)
  BIND(LANG(?itemLabel) AS ?lang)

  OPTIONAL { ?item schema:description ?dIt FILTER(LANG(?dIt)="it") }
  OPTIONAL { ?item schema:description ?dEn FILTER(LANG(?dEn)="en") }
  BIND(COALESCE(?dIt, ?dEn, "") AS ?itemDescription)
}
ORDER BY ?num
LIMIT 50
