1.1 Related Agents

Datatype: Relationship

CA element code: ca_entities

Quicktip: Agent responsibility or agent relations. Choose a relationship to an agent and then select the appropriate relator code. Once the relationship has been created the type of responsibility (primary, secondary etc.) can be specified by clicking on the 📎 paperclip that appears in the relationship. The paperclip in relationships gives access to interstitial information that gives additional information about the relationship.

1.4 Related Resources

Datatype: Relationship

CA element code: ca_objects

Quicktip: If a resource is part of a collection (or series, or series) you can link it via this relationship to the collection already entered in the system or that you wish to add via quick add. Use the relation 'Is part of' for linking. The inverse relation 'includes/contains' is not used here.
