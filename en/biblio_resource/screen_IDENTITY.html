1.5 Resource type (ethnomusicology, film music)

Datatype: InformationService

CA element code: typeOfEthnoFMRes_ME

Quicktip: Linked Data field. Indicate the type of resource (e.g., "Cue Sheet," "Field recording"). Enter at least three letters for Linked Data search and choose from the proposed alternatives. See documentation (i) for the list of allowed terms.

Linked Data field: for the allowed terms, updated on , click the following link to run the query in real time: Run Query (for complex queries, wait several seconds before results are returned).

SELECT DISTINCT ?item ?itemLabel ?itemDescription ?lang
WHERE {
    ?item wdt:P5008 wd:Q120325256.
    ?item wdt:P2579 ?discipline .
    VALUES ?discipline {wd:Q208365 wd:Q164204}
    ?item wdt:P279 wd:Q49848 .
    ?item(rdfs:label|skos:altLabel) ?itemLabel.
    FILTER(REGEX(?itemLabel, "^(***PLACEHOLDER***|***PLACEHOLDER*** |[^ ]+ ***PLACEHOLDER***)", "i"))
    FILTER(LANGMATCHES(LANG(?itemLabel), "it") || LANGMATCHES(LANG(?itemLabel), "en"))
    ?item schema:description ?itemDescription.
    FILTER(LANGMATCHES(LANG(?itemDescription), "it") || LANGMATCHES(LANG(?itemDescription), "en"))
    FILTER(LANG(?itemLabel) = LANG(?itemDescription))
   
  BIND(LANG(?itemLabel) AS ?lang)
  SERVICE wikibase:label { bd:serviceParam wikibase:language "[AUTO_LANGUAGE], en, it". }  

} 
order by ?item

1.7 Content language (ISO 639-3)

Datatype: InformationService

CA element code: cont_lang_ME

Repeatability: Yes

Quicktip: Linked Data field.Indicate the main language or languages ​​of the content.Start the Linked Research given by entering at least three letters using the ISO language code (e.g. 'kmr' for the northern Kurdish) or the name language ('Ital' for 'Italian' or 'language of Italian signs').Register all the languages ​​of the content, or record at least the main language and indicate that the resource is multilingual in the next field.Repeatable element.

Linked Data field: for the allowed terms, updated on , click the following link to run the query in real time: Run Query (for complex queries, wait several seconds before results are returned).

SELECT DISTINCT ?item ?itemLabel ?itemDescription ?value ?lang
WHERE {
  {
   ?item wdt:P220 ?value;
     rdfs:label ?itemLabel.
 FILTER((LANG(?itemLabel)) = "it") 
  FILTER(REGEX(?itemLabel, "^(***PLACEHOLDER***|[^ ]+ ***PLACEHOLDER***)", "i"))
  }
UNION
 {
   ?item wdt:P220 ?value;
     rdfs:label ?itemLabel.
 FILTER((LANG(?itemLabel)) = "en") 
  FILTER(REGEX(?itemLabel, "^(***PLACEHOLDER***|[^ ]+ ***PLACEHOLDER***)", "i"))
  }
  UNION
  {
    ?item wdt:P220 ?value .
  FILTER (REGEX(?value, "***PLACEHOLDER***", "i"))  
    }  
    SERVICE wikibase:label { bd:serviceParam wikibase:language "it, en". }
BIND(LANG(?itemLabel) AS ?lang)
 }
