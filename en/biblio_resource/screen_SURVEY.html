1.1 Record status information

Datatype: Information

CA element code: status

Quicktip: Indicate whether the card is a draft or is approved. Draft data are not displayed on the navigation portal.

1.2 Record data access information

Datatype: Information

CA element code: access

Quicktip: The access field has three possible values. The value "accessible with limitations" provides for anonymized publication of agent data and the use of only 'display' fields (E.g., for musical instruments: "Manufacture display") when present in the data structure.
