1.2.2 Language of title

Datatype: InformationService

CA element code: tit_lang2_ME

Quicktip: Linked data field. Specifies the language in which the title of the event is expressed, according to standard ISO 639-2 or ISO 639-3 codes. Search by language label (e.g., "ital" for Italian, or by ISO code,) is possible. Essential to support multilingual functions and disambiguation in international or digital environments.

Linked Data field: for the allowed terms, updated on , click the following link to run the query in real time: Run Query (for complex queries, wait several seconds before results are returned).

SELECT DISTINCT ?item ?itemLabel ?itemDescription ?value ?lang
WHERE {
  {
   ?item wdt:P220 ?value;
     rdfs:label ?itemLabel.
 FILTER((LANG(?itemLabel)) = "it") 
  FILTER(REGEX(?itemLabel, "^(***PLACEHOLDER***|[^ ]+ ***PLACEHOLDER***)", "i"))
  }
UNION
 {
   ?item wdt:P220 ?value;
     rdfs:label ?itemLabel.
 FILTER((LANG(?itemLabel)) = "en") 
  FILTER(REGEX(?itemLabel, "^(***PLACEHOLDER***|[^ ]+ ***PLACEHOLDER***)", "i"))
  }
  UNION
  {
    ?item wdt:P220 ?value .
  FILTER (REGEX(?value, "***PLACEHOLDER***", "i"))  
    }  
    SERVICE wikibase:label { bd:serviceParam wikibase:language "it, en". }
BIND(LANG(?itemLabel) AS ?lang)
 }

1.2.3 Additional language code

Datatype: List

CA element code: record_res_add_lang_code2_ME

Quicktip: Enter here the additional language codes provided by the SBN standard. These codes (three-letter alphanumeric, integrated ISO 639-2) allow you to specify up to three languages present in the resource, in order of predominance. If more than three languages are present, enter the predominant language code followed by MUL (multilingual). Also use conventional SBN codes for: MIS (uncoded languages), UND (unidentifiable language), ABS (resource without language content).
