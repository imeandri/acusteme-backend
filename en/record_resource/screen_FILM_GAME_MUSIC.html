1.5.1 Dramatic category

Datatype: InformationService

CA element code: dramaticcategory_ind_ME

Quicktip: Linked Data field. Select the dramatic category corresponding to the narrative or emotional function of the music resource in the context of photoplay music (e.g., "Chase," "Suspense," "Tenderness," "Villainy"). Values are to be chosen from controlled vocabulary, possibly mapped with external resources such as Wikidata or historical directories.

SELECT DISTINCT ?item ?itemLabel ?itemDescription ?lang
WHERE {
  ?item wdt:P31 wd:Q126077457.
    
?item (rdfs:label|skos:altLabel) ?itemLabel.
      FILTER(LANGMATCHES(LANG(?itemLabel), "it") || LANGMATCHES(LANG(?itemLabel), "en"))
    FILTER(REGEX(?itemLabel, "^(***PLACEHOLDER***|[^ ]+ ***PLACEHOLDER***)", "i"))
    
OPTIONAL {?item schema:description ?itemDescription.
    FILTER(LANGMATCHES(LANG(?itemDescription), "it") || LANGMATCHES(LANG(?itemDescription), "en"))}
    FILTER(LANG(?itemLabel) = LANG(?itemDescription))
   BIND(LANG(?itemLabel) AS ?lang)
}
ORDER BY ?item
LIMIT 100
