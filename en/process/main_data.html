1.3.1 Generic name of the process (Linked Data)

Datatype: InformationService

CA element code: process_denomination

Quicktip: Linked data field. Controlled terms that identify the general type of music process involved, according to structured vocabularies or linked data. Links the described practice to a standardized concept to ensure semantic interoperability.

Linked Data field: for the allowed terms, updated on , click the following link to run the query in real time: Run Query (for complex queries, wait several seconds before results are returned).

select ?item ?itemLabel ?itemDescription ?lang
where {
   SERVICE wikibase:mwapi {
      bd:serviceParam wikibase:endpoint "www.wikidata.org";
                      wikibase:api "EntitySearch";
                      mwapi:search "***PLACEHOLDER***";
                      mwapi:language "it".
      ?item wikibase:apiOutputItem mwapi:item.
      ?num wikibase:apiOrdinal true.
  }
  SERVICE wikibase:label {
                   bd:serviceParam wikibase:language "it, en" .
                   }
 
    ?item (rdfs:label|skos:altLabel) ?itemLabel.
  
     OPTIONAL {?item schema:description ?itemDescription.
                         FILTER(LANGMATCHES(LANG(?itemDescription), "it") || LANGMATCHES(LANG(?itemDescription), "en"))}
    FILTER(LANGMATCHES(LANG(?itemLabel), "it") || LANGMATCHES(LANG(?itemLabel), "en"))
    FILTER(LANG(?itemLabel) = LANG(?itemDescription))
    BIND(LANG(?itemLabel) AS ?lang)


}
order by ?item

1.3.2 Process dates

Datatype: DateRange

CA element code: processdates

Quicktip: 'daterange' field, enter single dates. Examples: 'August 30, 1746', '1900', 'June 7, 2007', 'June 2016'. Collective access also supports date-range and uncertain date expressions in date-range fields, but it is advisable to use bibliographic standards, which express uncertain ranges through the DATA 1-DATA2 method (see documentation).
