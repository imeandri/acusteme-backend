1.1 Production date

Datatype: DateRange

CA element code: filmcomp_date_ME

Quicktip: Specifies the official production date of the film. Can be national or international. Field 'daterange', enter individual dates. Examples: 'August 30, 1746', '1900', 'June 7, 2007', 'June 2016'. Collective access also supports date-range and uncertain date expressions in date-range fields, but it is advisable to use bibliographic standards, which express uncertain ranges through the DATA 1-DATA2 method (see documentation).

1.4.2 Film (IMDB from Wikidata)

Datatype: InformationService

CA element code: film_imdb_ind_ME

Quicktip: Linked Data field that allows linking to the Wikidata record of the film, leveraging linking via IMDb ID. Enter at least three letters of the title to activate the search engine.

Linked Data field: for the allowed terms, updated on , click the following link to run the query in real time: Run Query (for complex queries, wait several seconds before results are returned).

SELECT DISTINCT ?item ?itemLabel ?itemDescription ?lang ?imdbID ?IMDbLink
WHERE {
 
  ?item wdt:P345 ?imdbID;
        rdfs:label ?itemLabel.
 
  FILTER((LANG(?itemLabel)) = "en")
  FILTER(REGEX(?itemLabel, "^(***PLACEHOLDER***|[^ ]+ ***PLACEHOLDER***)", "i"))

 
  ?item wdt:P31 wd:Q11424.

  
  OPTIONAL {
    ?item schema:description ?itemDescription.
    FILTER(LANGMATCHES(LANG(?itemDescription), "en"))
  }
   BIND(LANG(?itemLabel) AS ?lang)

BIND(
  CONCAT("IMDb: ", "https://www.imdb.com/it/title/", ?imdbID) AS ?IMDbLink)
}
ORDER BY ?item
LIMIT 100
