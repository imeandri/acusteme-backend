1.4 Audio/video segmentation

Datatype: Container

CA element code: LOD_segmentator_CN

Repeatability: Yes

Quicktip: Repeatable container to segment audio/audiovisual resources: each row describes an occurrence by linking a LOD concept to a time code (start/end) and may include a user-friendly segment title. If the same concept recurs at different points, repeat the row with different time codes. Use “et passim” when the occurrence is scattered/intermittent and no precise segmentation is performed (optionally providing a broader enclosing interval or a note).

1.4.2 Ethnomusicological/Ethnographic descriptor (LOD)

Datatype: InformationService

CA element code: LOD_concept_ME

Quicktip: Linked Data field. Indicate the LOD concept.

Linked Data field: for the allowed terms, updated on , click the following link to run the query in real time: Run Query (for complex queries, wait several seconds before results are returned).

select distinct ?item ?itemLabel ?itemDescription ?lang 
where {
   SERVICE wikibase:mwapi {
      bd:serviceParam wikibase:endpoint "www.wikidata.org";
                      wikibase:api "EntitySearch";
                      mwapi:search "***PLACEHOLDER***";
                      mwapi:language "it".
      ?item wikibase:apiOutputItem mwapi:item.
      ?num wikibase:apiOrdinal true.
  }
  SERVICE wikibase:label {
                   bd:serviceParam wikibase:language "[AUTO_LANGUAGE],it,en" .
                   }
  ?item (rdfs:label|skos:altLabel) ?itemLabel.
  OPTIONAL {
     ?item schema:description ?itemDescription.
      FILTER(LANGMATCHES(LANG(?itemDescription), "it") || LANGMATCHES(LANG(?itemDescription), "en"))
   }
  FILTER(LANGMATCHES(LANG(?itemLabel), "it") || LANGMATCHES(LANG(?itemLabel), "en"))
      FILTER(LANG(?itemLabel) = LANG(?itemDescription))
    BIND(LANG(?itemLabel) AS ?lang)
}

Quicktip: Indicates the starting point of the audiovisual segment, expressed in timecode format (e.g., 00:05:12 for 5 minutes and 12 seconds). Used to define the time portion of the document to be isolated, described or analyzed.
