1.3.1 Categoria drammatica

Datatype: InformationService

CA element code: dramaticcategory_ME

Quicktip: Campo Linked Data. Termini controllati che identificano la Categoria Drammatica del brano quando espressa dal titolo o da elementi del titolo relativi all'intenzione semantico-affettiva del brano ed espressa dall'autore o dall'editore di collezioni di musica per film. Ad esempio, "Misterioso-Pizzicato, Storm, Orientale..."

Campo Linked Data: Per i termini ammissibili, aggiornati al , clicca sul seguente link per eseguire la query in tempo reale: Esegui Query (attenzione: per query complesse occorre attendere diversi secondi prima della restituzione dei risultati).

SELECT DISTINCT ?item ?itemLabel ?itemDescription
WHERE {
  # Selezioniamo gli elementi che sono istanze di Q126077457
  ?item wdt:P31 wd:Q126077457.
  
  # Controlliamo etichette e alias per la stringa "Cabiria"
  {
    ?item rdfs:label ?itemLabel.
    FILTER((LANG(?itemLabel)) = "en")
    FILTER(REGEX(?itemLabel, "^(***PLACEHOLDER***|[^ ]+ ***PLACEHOLDER***)", "i"))
  }
  UNION
  {
    ?item skos:altLabel ?itemLabel.
    FILTER((LANG(?itemLabel)) = "en")
    FILTER(REGEX(?itemLabel, "^(***PLACEHOLDER***|[^ ]+ ***PLACEHOLDER***)", "i"))
  }

  # Opzionalmente, otteniamo la descrizione se disponibile
  OPTIONAL {
    ?item schema:description ?itemDescription.
    FILTER(LANGMATCHES(LANG(?itemDescription), "en"))
  }
}
ORDER BY ?item
LIMIT 100

1.3.3 Tonalità (IFLA da Wikidata)

Datatype: InformationService

CA element code: cuetonalityinc_ME

Quicktip: Campo Linked Data. Seleziona la tonalità dell’opera musicale utilizzando i termini standardizzati derivati dal Thesaurus IFLA per le tonalità (in formato URI), con corrispondenze in Wikidata. Questo campo garantisce l’allineamento semantico tra catalogazione musicale e modelli internazionali (es. UNIMARC, RDA).

Campo Linked Data: Per i termini ammissibili, aggiornati al , clicca sul seguente link per eseguire la query in tempo reale: Esegui Query (attenzione: per query complesse occorre attendere diversi secondi prima della restituzione dei risultati).

SELECT distinct ?item ?itemLabel ?itemDescription ?value  
WHERE {
  {?item wdt:P9112 ?value;
        rdfs:label ?itemLabel.
   FILTER (REGEX(?value, "key#", "i"))
 FILTER((LANG(?itemLabel)) = "it") 
 FILTER(REGEX(?itemLabel, "^(***PLACEHOLDER***|[^ ]+ ***PLACEHOLDER***)", "i"))
 OPTIONAL {
    ?item schema:description ?itemDescription.
    FILTER(LANGMATCHES(LANG(?itemDescription), LANG(?itemLabel)))
  }
  }
UNION
 {
   ?item wdt:P9112 ?value;
     rdfs:label ?itemLabel.
   FILTER (REGEX(?value, "key#", "i"))
 FILTER((LANG(?itemLabel)) = "en") 
 FILTER(REGEX(?itemLabel, "^(***PLACEHOLDER***|[^ ]+ ***PLACEHOLDER***)", "i"))
 OPTIONAL {
    ?item schema:description ?itemDescription.
    FILTER(LANGMATCHES(LANG(?itemDescription), LANG(?itemLabel)))
  }
  }
        SERVICE wikibase:label { bd:serviceParam wikibase:language "it, en". }

 }

ORDER BY ?item
