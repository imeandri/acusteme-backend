1.4 Segmentazione audio/video

Datatype: Container

CA element code: LOD_segmentator_CN

Ripetibilità: Sì

Quicktip: Container ripetibile per segmentare risorse audio/audiovisive: ogni riga descrive un’occorrenza con concetto LOD e timecode (inizio/fine) e può avere un titolo user-friendly. Se lo stesso concetto ricorre in punti diversi, ripetere la riga con timecode differenti. Usare “et passim” quando la ricorrenza è diffusa/intermittente e non si procede a segmentazione puntuale (eventualmente indicando un intervallo contenitore o una nota).

1.4.2 Descrittore etnomusicologico/etnografico (LOD)

Datatype: InformationService

CA element code: LOD_concept_ME

Quicktip: Campo Linked Data. Indicare il concetto LOD.

Campo Linked Data: Per i termini ammissibili, aggiornati al , clicca sul seguente link per eseguire la query in tempo reale: Esegui Query (attenzione: per query complesse occorre attendere diversi secondi prima della restituzione dei risultati).

select distinct ?item ?itemLabel ?itemDescription ?lang 
where {
   SERVICE wikibase:mwapi {
      bd:serviceParam wikibase:endpoint "www.wikidata.org";
                      wikibase:api "EntitySearch";
                      mwapi:search "***PLACEHOLDER***";
                      mwapi:language "it".
      ?item wikibase:apiOutputItem mwapi:item.
      ?num wikibase:apiOrdinal true.
  }
  SERVICE wikibase:label {
                   bd:serviceParam wikibase:language "[AUTO_LANGUAGE],it,en" .
                   }
  ?item (rdfs:label|skos:altLabel) ?itemLabel.
  OPTIONAL {
     ?item schema:description ?itemDescription.
      FILTER(LANGMATCHES(LANG(?itemDescription), "it") || LANGMATCHES(LANG(?itemDescription), "en"))
   }
  FILTER(LANGMATCHES(LANG(?itemLabel), "it") || LANGMATCHES(LANG(?itemLabel), "en"))
      FILTER(LANG(?itemLabel) = LANG(?itemDescription))
    BIND(LANG(?itemLabel) AS ?lang)
}
